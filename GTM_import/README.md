# GTM Import

This folder contains different types of files to be imported in Google Tag Manager (GTM) to install Nantu.

## Files

### NantuTestSuiteFull.json
Contains most of the Nantu modules including:
- **Nantu Tracking** - Core tracking functionality
- **Nantu Target Helper** - Targeting assistance tools
- **Nantu QA Tab** - Quality assurance interface
- **Nantu QA Container** - QA container management
- **Nantu Clarity** - Analytics clarity features
- **Nantu A/B Test Template** - Template for A/B testing setup

Additionally includes built-in variables such as:
- **Page URL** - Current page URL tracking
- **Container ID** - GTM container identification

This comprehensive file provides a complete Nantu installation for GTM environments.
